# 📋 Weekly Report — 2026년 6월 16일

## 이번 주 한 줄 요약

> **Focus Matrix 앱의 첫 번째 버전을 완성하고 로컬 설치까지 마쳤습니다.**

---

## ✅ 완성된 기능

### 🗂 아이젠하워 매트릭스 화면

할 일을 4개의 분면에 나눠 한눈에 볼 수 있는 메인 화면을 만들었습니다.

- **Q1 Do Now** (빨강) — 오늘 당장 해야 할 일
- **Q2 Schedule** (파랑) — 중요하지만 급하지 않은 일
- **Q3 Delegate** (주황) — 급하지만 덜 중요한 일
- **Q4 Eliminate** (회색) — 하지 않아도 되는 일

카드를 **끌어다 다른 분면에 놓으면** 바로 이동됩니다.

---

### ➕ 작업 추가 / 수정 / 삭제

- `⌘N` 으로 어디서든 빠르게 새 작업을 추가할 수 있습니다
- 제목, 메모를 입력하고 **중요한지 / 급한지** 두 가지만 선택하면 분면이 자동으로 결정됩니다
- 카드를 클릭하면 수정, 우클릭하면 삭제할 수 있습니다

---

### 🕐 마감 일시 + 알림 설정

작업마다 마감 날짜와 시간을 함께 설정할 수 있습니다.

- 날짜와 시간을 동시에 선택 (기본값: 오늘 오후 6시)
- 알림을 몇 분 전에 받을지 선택 — 5분 / 10분 / 15분 / 30분 / 1시간 / 2시간 / 1일 전
- 알림이 정확히 몇 시에 오는지 미리보기로 확인 가능
- 마감이 지난 작업은 카드에서 **빨간색**으로 표시됩니다
- 알림이 설정된 작업에는 🔔 아이콘이 표시됩니다

> 알림 발송 기능은 다음 주에 연동할 예정입니다.

---

### ✔️ 완료 처리

- 카드의 동그라미를 클릭하면 작업이 완료됩니다
- 완료된 작업은 **Completed 탭**으로 자동 이동
- 오늘 완료 / 이번 주 완료 / 전체 보기로 필터링 가능
- 실수로 완료했다면 클릭 한 번으로 되돌릴 수 있습니다

---

### 📌 메뉴바 앱

- 화면 오른쪽 상단 메뉴바에 **Focus Matrix 아이콘**이 상시 표시됩니다
- 클릭하면 지금 당장 해야 할 Q1, Q2 작업을 바로 확인할 수 있습니다
- 메뉴바에서도 새 작업을 추가할 수 있습니다

# 📋 Weekly Report — 2026년 6월 23일

## 이번 주 한 줄 요약

> **작업 추가 흐름을 더 자연스럽게 개선하고, 마감 시간에 따라 앱이 스스로 우선순위를 조정하도록 만들었습니다.**

---

## ✅ 완성된 기능

### ➕ 분면에서 바로 작업 추가

각 분면에서 새 작업을 훨씬 빠르게 추가할 수 있게 되었습니다.

- 각 분면 하단에 **`+ 작업 추가`** 버튼이 항상 표시됩니다
- 버튼을 누르면 해당 분면의 중요도/긴급도가 **미리 선택된 채로** 모달이 열립니다
- 예: Q1 분면에서 추가하면 '중요함 + 긴급함'이 자동 체크된 상태로 시작

---

### ⚡ 마감 24시간 이내 자동 긴급 전환

마감이 다가오는 작업을 앱이 스스로 감지해 분면을 이동시킵니다.

- 마감까지 **24시간 이내**가 되면 자동으로 긴급 분면으로 전환
  - Q2 (중요·여유) → **Q1 (중요·긴급)**
  - Q4 (여유·덜 중요) → **Q3 (긴급·덜 중요)**
- 자동 전환된 카드에는 **⚡ 24h** 뱃지가 표시됩니다
- 앱 실행 시 즉시 확인하고, 이후 1분마다 자동으로 재체크
- 직접 드래그해서 이동하면 뱃지가 사라집니다
